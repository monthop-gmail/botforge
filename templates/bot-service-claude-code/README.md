## Quick Start

```bash
cp .env.example .env
# Edit .env with your Anthropic API key, LINE credentials, tunnel token

docker compose up -d --build

# Webhook URL: https://{{DOMAIN}}/webhook
```

## Architecture

```
LINE app → Cloudflare Tunnel → line-bot (Bun, port 3000) → server (Hono + Agent SDK, port 4096) → Anthropic API
```

**3 Docker Services:**
- **server** — Claude Code API server (Hono + Claude Agent SDK)
- **line-bot** — LINE webhook handler (Bun/TypeScript)
- **cloudflared** — Cloudflare tunnel

## Commands

```bash
docker compose up -d --build                    # Deploy
docker logs {{CONTAINER_PREFIX}}-line-bot -f     # Bot logs
docker logs {{CONTAINER_PREFIX}}-server -f       # Server logs
docker compose up -d --build line-bot            # Rebuild bot only
```

## LINE Bot Commands

| Command | Description |
|---------|-------------|
| `/new` | Start new session |
| `/abort` | Cancel current prompt |
| `/sessions` | View session info + cost |
| `/cost` | Show total cost |

## Models

| Model | Description |
|-------|-------------|
| `sonnet` | Balanced (default) |
| `opus` | Strongest quality |
| `haiku` | Fast, cheapest |

Set via `CLAUDE_MODEL` in `.env`
