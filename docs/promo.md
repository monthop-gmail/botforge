## Reddit / Dev.to / GitHub Discussions (Technical)

### English

```
Botforge: One command to create AI-powered LINE Bots

I built a CLI tool that generates complete LINE Bot projects with AI integration out of the box. No boilerplate, no hours of config.

How it works:

git clone https://github.com/monthop-gmail/botforge.git
cd botforge
./botforge new my-bot
cd projects/my-bot/bot-service
cp .env.example .env   # add your API keys
docker compose up -d --build

That's it. Your bot is live.

Two AI engines:

| | OpenCode | Claude Code |
|---|---|---|
| Models | 40+ (Claude, GPT, Gemini, DeepSeek, Qwen) | Claude (Sonnet/Opus/Haiku) |
| Setup | More config, more flexibility | Simple, just ANTHROPIC_API_KEY |
| Cost control | Per-provider | Built-in budget limit |

Tech stack: Bun, TypeScript, Docker Compose, Cloudflare Tunnel

License: MIT — free forever, self-hosted, no lock-in

Looking for contributors! Whether it's new engines, templates, docs, or bug fixes — PRs welcome.

Repo: https://github.com/monthop-gmail/botforge
```
