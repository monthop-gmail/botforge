## Architecture

```
LINE app → Cloudflare Tunnel → line-bot (Bun, port 3000) → adkcode (Python/FastAPI, port 8000) → Gemini
```

**3 Docker Services:**
- **server** — ADKcode multi-agent system (orchestrator → coder, reviewer, tester)
- **line-bot** — LINE webhook handler (Bun/TypeScript)
- **cloudflared** — Cloudflare tunnel (exposes webhook to internet)

## Commands

## LINE Bot Commands

| Command | Description |
|---------|-------------|
| `/new` | Start new session |
| `/abort` | Cancel current prompt |
| `/sessions` | View session status |
| `/about` | About this bot |
| `/help` | Show all commands |

## Multi-Agent System

| Agent | Model | Role |
|-------|-------|------|
| orchestrator | gemini-2.5-flash | Routes tasks to sub-agents, web search |
| coder | gemini-2.0-flash | Write, edit, create code files |
| reviewer | gemini-2.5-flash | Code review, bug detection |
| tester | gemini-2.0-flash | Run tests, fix failures |

## Plugins

Built-in plugins in `server/plugins/`:

- **data** — data analysis, SQL queries, visualization, dashboards
- **productivity** — task management, memory management
