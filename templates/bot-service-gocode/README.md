```bash
# 3. Done! Webhook URL:
# https://{{DOMAIN}}/webhook
```

## Architecture

```
LINE app → Cloudflare Tunnel → line-bot (Bun, port 3000) → gocode (Go, port 4096) → LLM
```

**3 Docker Services:**
- **server** — Gocode AI coding agent (Go, OpenAI-compatible LLM, built-in tools)
- **line-bot** — LINE webhook handler (Bun/TypeScript)
- **cloudflared** — Cloudflare tunnel (exposes webhook to internet)

## Commands

```bash
# Deploy
docker compose up -d --build

# Logs
docker logs {{CONTAINER_PREFIX}}-line-bot --tail 30
docker logs {{CONTAINER_PREFIX}}-server --tail 30

# Rebuild only bot
docker compose up -d --build line-bot
```

## LINE Bot Commands

| Command | Description |
|---------|-------------|
| `/new` | Start new session |
| `/abort` | Cancel current prompt |
| `/sessions` | View session status |
| `/about` | About this bot |
| `/help` | Show all commands |
